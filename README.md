libre-etherpad
==============

Etherpad image with bundled plugins.

**Installed plugins:**
- ep_adminpads2
- ep_desktop_notifications
- ep_announce
- ep_themes_ext
- ep_inline_voting
- ep_prometheus
- ep_message_all
- ep_mypads
